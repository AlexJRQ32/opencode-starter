# Skill: cursos-gratis

Skill para buscar y recomendar cursos GRATIS de programacion, tecnologia y desarrollo.

**Nivel A (prioridad):** freeCodeCamp, The Odin Project, MDN, W3Schools, Khan Academy, Harvard CS50, MIT OCW, Google Devs, Microsoft Learn, Roadmap.sh

**Nivel B (YouTube):** midudev, MoureDev, HolaMundo, Fazt, Dalto, Carlos Azaustre, Develoteca, NetNinja, Traversy Media, freeCodeCamp, Fireship, Web Dev Simplified, Kevin Powell

**Nivel C (gratuito limitado):** Coursera audit, edX audit, Udemy gratis, Platzi gratis, Codecademy basico, Scrimba gratis

**Proceso:** Preguntar tema + nivel + idioma + formato -> buscar en A -> B -> C -> recomendar con enlace directo.

**Prohibido:** Cursos de paga, LinkedIn Learning, Datacamp pago, canales no verificados.
